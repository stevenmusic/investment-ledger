單股損益排行
還沒有持倉資料。到 持倉 分頁看看怎麼新增。
目前持倉
新增或修改持股：到 repo 裡編輯 data/holdings.json（GitHub 網頁上就能直接編輯存檔），存檔後等下一次排程（最多約30分鐘）或手動觸發 Action，Dashboard 就會更新。第一次被系統抓到的平均成本會自動被鎖定為跌幅通知的基準價，之後加碼不會改變通知門檻。
觀察清單
新增觀察標的：編輯 data/watchlist.json，加一筆並填 targetPrices（想被通知的價位，可以填多個）。
通知設定

開啟後，這支手機/瀏覽器會在持股跌 5% / 10% / 15%，或 Watchlist 標的跌到目標價時收到推播通知。

關於這個工具

所有股價與通知檢查都由 GitHub Actions 排程處理，資料存在這個 repo 裡的 JSON 檔案中，沒有額外的伺服器或資料庫。原始碼： 查看 repo
